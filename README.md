# The-Way-He-Wants-Me-Mouth-Full-Of-Static-by-N.A.
I woke up with dried blood under my fingernails and no memory of how it got there. They say I'm sick. Dangerous. Broken. Dr. Korrin says he can fix me. All I have to do is stop fighting, surrender my memories, and trust the machine that keeps pulling me deeper into myself. The problem is, the more I remember, the less I want to be saved.
